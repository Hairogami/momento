📋 Flow final — 5 étapes

  1. Explore — libre, anonyme, gate au clic profil
  2. Signup — modal avec 4 value props du dashboard
  3. Onboarding — type événement + catégories prestas
  4. Budget — répartition par catégorie, feedback réaliste/serré/ambitieux, insight marché
  5. Swipe prestas — 2-3 par catégorie, filtrés par budget, avec prix affiché
  6. Dashboard — 35% prêt, prestas + budget + tâches personnalisées
Le cercle vertueux : Budget éduque le client → swipe filtré par budget → prestas shortlistés → dashboard Free avec FOMO (widgets Pro en aperçu) → client veut contacter → paywall Pro → conversion.
