📱 Comment l'utilisateur l'installe

🤖 Android (Chrome) — 85% MA

Automatique. Après 2-3 visites, Chrome affiche "Installer Momento" en bas de l'écran. 1 tap → icône sur home screen. Push notifs supportées nativement.

🍎 iOS (Safari) — 15% MA

Manuel. Bouton "Partager" → "Sur l'écran d'accueil". Pas de prompt auto (restriction Apple). Push notifs OK depuis iOS 16.4 (2023).

💻 Desktop (Chrome/Edge)

Automatique. Icône "Installer" dans la barre d'URL. Ouvre Momento en fenêtre dédiée, sans onglets — utile pour les prestataires qui gèrent leurs leads.

⚙️ Mise en place technique (Next.js 16)

1
Créer public/manifest.json

Déclare le nom, les icônes, la couleur de thème (#a78bfa), le mode standalone. C'est le fichier qui dit au navigateur "cette app est installable".

2
Générer les icônes Momento

3 formats : icon-192.png + icon-512.png + apple-touch-icon.png (180×180). Outil recommandé : realfavicongenerator.net (1 upload → tous les formats).

3
Installer @ducanh2912/next-pwa

Plugin Next.js qui génère automatiquement le service worker (cache offline, stratégies de revalidation). Compatible Next.js 16.

4
Ajouter metadata dans app/layout.tsx

Une ligne dans les metadata Next.js pour déclarer le manifest + couleur de thème + apple-touch-icon.

5
Tester avec Lighthouse

Chrome DevTools → Lighthouse → cocher "PWA" → Run. Score 90+ requis pour que le prompt "Installer" s'affiche. Checklist : HTTPS ✓ (Vercel), manifest ✓, service worker ✓, icônes ✓.

6
Déployer sur Vercel

HTTPS auto = condition PWA remplie. Le service worker s'enregistre au 1er chargement. Les visiteurs existants deviennent éligibles à l'installation dès leur prochaine visite.

🎁 Ce que le PWA débloque pour Momento

📴 Offline mode

Les pages visitées restent accessibles sans réseau. Utile dans les salles de fêtes au Maroc où la 4G galère.

⚡ Lancement instantané

Pas de barre d'URL, pas de tabs, ouverture directe = ressenti "vraie app". Splash screen aux couleurs Momento.

🔔 Push notifications

Android full support, iOS 16.4+. "Studio Anass a répondu" → l'utilisateur revient. Retention x3.

🏠 Icône brand quotidienne

Yasmine voit Momento chaque fois qu'elle déverrouille son tel. Top of mind pendant 6 mois de préparation mariage.

⚠️ Limites PWA (vs Capacitor)

Pas de stores Pas de présence App Store / Play Store. Utilisateurs doivent tomber sur momentoevents.app d'abord.

iOS restrictif Pas de prompt auto, installation moins intuitive (Partager → Écran accueil), certaines API limitées.

Pas de paiements in-app Si tu veux vendre Pro (200 MAD/mois) via App Store (30% fee Apple), il faut Capacitor.

Stratégie recommandée : PWA maintenant (1 semaine) → valide l'usage mobile → Capacitor dans 2-3 mois pour publier sur les stores une fois la traction prouvée.
