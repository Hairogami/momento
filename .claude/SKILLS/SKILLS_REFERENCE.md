## Loops utiles
```
/loop gsd-code-review          # review après chaque commit
/loop vercel:status            # surveiller un déploiement
/loop gsd-scan                 # scan sécurité récurrent
/loop gsd-health               # santé projet
/loop gsd-next                 # enchaîner les tâches
/review-loop                   # review continue pendant dev
```

## Meta-skills (triggers proactifs)

### `skill-creator` — trigger AUTO
Dès qu'un workflow réussi de 3+ étapes a des chances d'être refait → proposer de le figer en skill.

### `claude-automation-recommender` — trigger DIFFÉRÉ
En fin de session seulement. Si 3+ actions identiques répétées → proposer hook/loop/skill.

- **Chrome DevTools** : perf/a11y live (LCP, accessibilité, console/network)
- **Vercel** : deploy + logs + env + search docs
