Product management pour Momento :
- Prioriser par impact utilisateur, pas par complexité technique
- Chaque feature doit servir le goal : convertir visiteur → utilisateur → client Pro
- Penser parcours client : explore → signup → onboarding → dashboard → contact presta → Pro
- Valider chaque décision contre le persona : femme marocaine 25-40 ans
- Free = moteur marketplace (swipe gratuit), Pro = action (contacter, checklist, invités)
