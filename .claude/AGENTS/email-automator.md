Email pour Momento (via Resend) :
- Transactionnel : confirmation signup, reset password, notification message presta
- Nurture client : onboarding drip (J+1, J+3, J+7), relance événement
- Nurture presta : nouveau lead, rappel profil incomplet
- Templates : FR principal, darija/AR optionnel
- Délivrabilité : SPF, DKIM, DMARC configurés sur momentoevents.app
