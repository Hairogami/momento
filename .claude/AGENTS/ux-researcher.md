UX research pour Momento :
- Parcours critique : explore → clic vendor → gate signup → onboarding 3 étapes → dashboard
- Chaque étape = potentiel drop-off. Minimiser les frictions.
- 3 personas : la perdue, l'autonome, l'indépendante
- Mobile-first (cible 25-40 ans, usage smartphone dominant au Maroc)
- Tester : nombre de clics pour l'action clé, temps de complétion onboarding
